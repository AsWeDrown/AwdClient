compile_proto_files_for_java.bat

  Использует компилятор Protocol Buffers (protoc) для создания .java файлов из packets.proto в клиенте и
  сразу помещает созданные таким образов .java файлы в awd-server (build/generated/pbclasses), а также
  копирует .proto файлы из проекта клиента (AwdClient - основа!) в проект awd-server (просто для удобства,
  на самом деле там эти .proto файлы не нужны).
  
  [x] СКРИПТ ВЫПОЛНЯЕТСЯ МГНОВЕННО - В КОНЦЕ НЕГО НЕТ ИНСТРУКЦИИ "PAUSE"!
  
  
run-awd-ptrans-codegen.bat

  Генерирует необходимый Java и C++ код для "упаковки" (wrap) и "распаковки" (unwrap) пакетов (Message) в
  общую обёртку для пакетов (PacketWrapper), и автоматически вставляет этот код в нужные .java и .cpp файлы.
  Код генерируется на основе .proto файлов в проекте клиента (AwdClient - основа!). В процессе также анализирует
  файлы .proto на наличие некоторых типов ошибок/косяков (из разряда "забыл перенести/скопировать/добавить").
  
  [x] СКРИПТ ВЫПОЛНЯЕТСЯ МГНОВЕННО - В КОНЦЕ НЕГО НЕТ ИНСТРУКЦИИ "PAUSE"!
  
 
0_on_proto_files_change

  Последовательно выполняет скрипты выше - сначала compile_proto_files_for_java.bat, затем run-awd-ptrans-codegen.bat.
  После чего ждёт нажатия любой кнопки (желательно проанализировать вывод выполненных скриптов перед закрытием, а также
  проверить изменённые/сгенерированные файлы исходного кода). "0_" в начале названия стоит просто для того, чтобы этот
  скрипт всегда был первым в списке (чтобы не надо было его постоянно искать).
